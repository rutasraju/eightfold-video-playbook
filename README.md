# Eightfold Design Concept Video Playbook

A shared standard for the videos Product and Design ship as a deliverable — concept finalization, vision exploration, and internal alignment. Produced by Product + Design. Reviewed by Leadership for sales-grade promotion. Adapted by Marketing for public distribution.

**→ Read the playbook: [index.html](./index.html)**

## Contents

| Chapter | What's inside |
| --- | --- |
| 00 — Scope | What counts as a concept video; what doesn't |
| 01 — Principles | One idea · honest fidelity · designed enough to elevate |
| 02 — Before you start | Name the room · name the ask |
| 03 — The seven beats | opportunity → thinking → concept → journey → intelligence → unlock → ask |
| 04 — Look & sound | Do / don't, language shifts, the fidelity chyron |
| 05 — The playback test | One watch, summary back |
| 06 — The three gates | One idea · honest fidelity · unmistakable ask |
| 07 — Lifecycle & handoff | Production → Promotion to demo → External adaptation, with RACI |

## How to use it

- Reference **Chapter 00** at the start of every project — confirm this is the right playbook before applying its rules.
- Reference **Chapter 06** (the three gates) before any video ships internally.
- Reference **Chapter 07** before promoting a video to demo use or handing assets to Marketing.
- Print **Chapter 04** and pin it above your desk.

## Ownership

- **Produced by** — Product + Design
- **Reviewed by** — Leadership (promotes to demo) → Marketing (adapts for public)
- **Owner** — UX Team (co-owned with Product)

Updates are versioned and announced via the Design org. This playbook covers concept videos only; marketing videos, sales demos, customer stories, and explainers each warrant their own playbook (forthcoming) and their own owners.
